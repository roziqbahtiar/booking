Petunjuk Penggunaan Sistem

  1. Lihat file Sampel Pengajuan yang sudah ada sebelum anda input
  2. Input pengajuan melalui menu Input Pengajuan
  3. Sesuaikan format pengajuan pada form yang tertampil sesuai dengan file Sampel Pengajuan
  4. Sistem akan mengirimkan email nomer tiket pengajuan setelah pengajuan berhasil terinput, harap disimpan nomer tiket pengajuan untuk pengecekan status
  5. Sistem akan mengirimkan email notifikasi bahwa pengajuan "Diterima" atau "Ditolak"
  6. Satu mahasiswa hanya diperkenankan menginputkan satu pengajuan dan pengajuan selanjutnya menunggu tiket pengajuan sebelumnya sudah dalam status "Diterima", "Ditolak", atau tiket pengajuan dibatalkan
  7. Form pembatalan tiket pegajuan melalui alamat http://skripsi.ptik.unnes.ac.id
